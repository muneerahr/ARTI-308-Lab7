# Logistic Regression Lab

## Overview
This project applies Logistic Regression on an advertising dataset.
## Steps
- Data exploration (EDA)
- Data preparation
- Train-test split
- Model training using Logistic Regression
- Model evaluation using classification report

## Tools Used
- Python
- Pandas
- Seaborn
- Scikit-learn

## Result
The model achieved good performance with high accuracy and balanced precision and recall.
